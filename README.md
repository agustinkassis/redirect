## Lightweight Redirect

Just fork it.

Copy **.env.sample** into **.env** file and set your destination URL.

```bash
VITE_DESTINATION="http://lacrypta.com.ar"
```

Deploy it and assign a subdomain.

### Features

- [x] Vite
- [x] Custom Title
- [ ] Analytics
- [x] Netlify Button
